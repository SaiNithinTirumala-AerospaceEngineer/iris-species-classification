## Problem statement

The Iris dataset is a canonical multi-class classification benchmark
introduced by Fisher (1936). This project trains and compares five
classifiers — K-Nearest Neighbours, Support Vector Machine, Decision
Tree, Random Forest, and Logistic Regression — evaluating accuracy,
precision, recall, and F1 score on held-out test data. Decision boundary
visualisation and cross-validated performance comparison identify the
optimal classifier for this three-class problem.

*Developed during the Bharat Intern Machine Learning Virtual Internship,
10 August – 10 September 2023.*
